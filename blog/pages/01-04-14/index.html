sudo cd

$sudo cd /root
sudo: cd: command not found

Now what? You have to get to those config files that belong to root but you’re not root and cd doesn’t exist. cd doesn’t exist?! BUT IT DOES EXIST! WHAT?!

The Why and the Fix

Sudo is used for running binaries as root temporarily. You can try searching for cd (id est "whereis cd"), but it doesn’t exist! That’s because cd is part of the shell and not of another program. In order to use cd as root, we must use the shell as a root. A solution is to su the way through and become the root user 

$sudo su
$cd /root

or explicitly use the shell as root temporarily 

$sudo bash
$cd /root
